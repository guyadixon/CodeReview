# Security Code Review Guides

This directory contains language-specific security code review guides for the seven programming languages covered by this project. Each guide provides manual review best practices, common security pitfalls, recommended SAST tools with setup instructions, language-specific vulnerability patterns, cross-references to the examples in this project, and a quick-reference checklist.

| Guide | Description |
|---|---|
| [Python](python_security_review_guide.md) | Python security code review guide covering Flask, Bandit, and Semgrep |
| [Java](java_security_review_guide.md) | Java security code review guide covering Spring Boot, SpotBugs, PMD, and Semgrep |
| [Go](go_security_review_guide.md) | Go security code review guide covering Gin, gosec, staticcheck, and Semgrep |
| [Rust](rust_security_review_guide.md) | Rust security code review guide covering Actix-web, cargo-audit, clippy, and Semgrep |
| [C](c_security_review_guide.md) | C security code review guide covering libmicrohttpd, cppcheck, clang-tidy, and Semgrep |
| [C++](cpp_security_review_guide.md) | C++ security code review guide covering cpp-httplib, cppcheck, clang-tidy, and Semgrep |
| [JavaScript](javascript_security_review_guide.md) | JavaScript/Node.js security code review guide covering Express.js, eslint-plugin-security, NodeJsScan, and Semgrep |
